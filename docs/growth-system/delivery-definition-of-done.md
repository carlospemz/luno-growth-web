# Definition of Done — Luno Growth Services

> Un entregable NO se considera terminado hasta que cumpla TODOS los criterios de su tipo.
> Los criterios marcados [ANGEL] requieren su aprobacion explicita.

---

## 1. WEBSITE — Definition of Done

### Express ($7,900)
- [ ] 1 pagina publicada con max 5 secciones
- [ ] Copy final en todas las secciones
- [ ] WhatsApp button + boton de llamada funcionando
- [ ] Mobile responsive (probado en 3+ dispositivos)
- [ ] Carga < 3s
- [ ] Sin errores de consola
- [ ] Dominio configurado (o URL temporal entregada)
- [ ] 1 ajuste menor ejecutado
- [ ] QA Website checklist pasada
- [ ] [ANGEL] Aprobacion visual final

### Pro ($13,900)
- [ ] Todo lo de Express +
- [ ] Hasta 9 secciones
- [ ] Copy persuasivo (no generico)
- [ ] Micro-animaciones (reveals, hovers)
- [ ] Preparada para Google (meta tags, H1s, schema si aplica)
- [ ] 1 ronda de ajustes completada
- [ ] Analytics basico configurado (GA4)
- [ ] QA Website + SEO checklists pasadas
- [ ] [ANGEL] Aprobacion visual y de copy

### Enterprise ($24,900)
- [ ] Todo lo de Pro +
- [ ] 3-5 paginas con estructura
- [ ] 2+ flujos de conversion funcionando
- [ ] Medicion basica configurada
- [ ] 2 rondas de ajustes completadas
- [ ] Soporte de lanzamiento (dia del go-live)
- [ ] QA completa (Website + SEO + Performance)
- [ ] [ANGEL] Aprobacion final de todo

## 2. MARKETING — Definition of Done

### Setup inicial
- [ ] Pilares de contenido definidos y aprobados
- [ ] Calendario del primer mes creado
- [ ] Plantillas de post creadas
- [ ] Accesos a redes sociales configurados
- [ ] [ANGEL] Estrategia aprobada

### Entrega mensual
- [ ] 8-12 piezas producidas
- [ ] Publicadas en fechas del calendario
- [ ] Metricas del mes documentadas
- [ ] Reporte mensual entregado
- [ ] Ajustes para el proximo mes propuestos
- [ ] [ANGEL] Reporte revisado

---

## 3. SEO — Definition of Done

### Setup inicial
- [ ] Keywords definidos por pagina
- [ ] Meta tags implementados
- [ ] Schema markup implementado (si Pro+)
- [ ] Sitemap.xml generado
- [ ] Google Search Console configurado
- [ ] Google My Business configurado (si local)
- [ ] Baseline de posiciones documentado

### Entrega mensual (si servicio continuo)
- [ ] Posiciones monitoreadas
- [ ] Ajustes de keywords si necesario
- [ ] Contenido nuevo publicado (si Enterprise)
- [ ] Reporte mensual de trafico y posiciones
- [ ] [ANGEL] Reporte revisado

---

## 4. PROPUESTA COMERCIAL — Definition of Done

- [ ] QA Propuesta checklist pasada
- [ ] [ANGEL] Pricing aprobado
- [ ] [ANGEL] Messaging aprobado
- [ ] Enviada al cliente por canal acordado

## 5. ESTADOS DE ENTREGA

| Estado | Significado | Quien mueve |
|--------|-----------|-------------|
| `scoping` | Brief recibido, definiendo alcance | Sistema (automatico al recibir brief) |
| `research` | Research pack en generacion | Research engine |
| `strategy` | Strategy pack en generacion | Strategy engine |
| `strategy_review` | Strategy listo, esperando aprobacion | Angel |
| `building` | En construccion (website/marketing/SEO) | Builder |
| `qa` | En revision de calidad | QA checklist |
| `revision` | Cliente pidio cambios | Builder |
| `delivered` | Entregado al cliente | Builder + Angel |
| `completed` | Cliente acepto, proyecto cerrado | Angel |
| `cancelled` | Cancelado antes de entregar | Angel |
