## 2. MESSAGING HIERARCHY

### Mensaje principal (H1)
- Lo que el cliente quiere que su audiencia entienda en 3 segundos.
- Debe incluir: que ofrece + a quien + por que es diferente.

### Mensajes secundarios
1. Confianza: (por que confiar en este negocio)
2. Accion: (que hacer ahora — CTA principal)
3. Prueba: (evidencia, testimonios, numeros)
4. Urgencia: (por que ahora y no despues)

### Taglines por seccion
| Seccion | Tagline | Objetivo |

| Portfolio/Evidencia | | Generar confianza |

## 5. SEO DIRECTION

### Keyword strategy
| Keyword principal | Pagina target | Intent |
|-------------------|--------------|--------|
| | Home | transactional |
| | {extra_page} | informational |

### On-page checklist
- [ ] Title tag con keyword principal + ubicacion
- [ ] Meta description con CTA y diferenciador
- [ ] H1 con keyword (natural, no forzado)
- [ ] Alt text en imagenes con keywords
- [ ] Schema markup: LocalBusiness (si aplica)
- [ ] Open Graph tags
- [ ] Sitemap.xml
- [ ] Robots.txt

### Local SEO (si aplica)
- [ ] Google My Business configurado
- [ ] NAP consistente (Name, Address, Phone)
- [ ] Categorias correctas en GMB
- [ ] Horarios actualizados
- [ ] Fotos en GMB
