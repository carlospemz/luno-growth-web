## 6. CRITERIOS DE SALIDA (Definition of Done)

### Post individual
- [ ] Copy revisado (tono del Strategy Pack)
- [ ] Visual coherente con brand (colores, tipografia)
- [ ] CTA claro en caption o visual
- [ ] Hashtags relevantes (5-15)
- [ ] Sin errores de ortografia
- [ ] Aprobado (si es pilar core o promo)

### Calendario mensual
- [ ] 8-12 piezas minimo
- [ ] Balance de pilares respetado
- [ ] Al menos 1 campana activa
- [ ] Metricas del mes anterior revisadas
- [ ] Ajustes basados en performance
